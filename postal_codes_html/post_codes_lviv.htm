Пошук по вулиці здійснюється лише для міст та СМТ.
Для здійснення пошуку по вулиці необхідно обов’язково зазначити назву населеного пункту. Якщо населений пункт є місто чи СМТ, стане доступним поле «Введіть назву вулиці».
Для того, щоби швидко знайти необхідну вулицю, варто вказувати перші 3-4 літери назви, не вводячипри цьому типу вулиці (проспект, провулок та інше). Зверніть увагу, що для деяких населених пунктів в дужках може бути вказана приналежність до сільської чи міської ради
Для здійснення пошуку за обласним центром необхідно обрати область, а у полі "Оберіть район" обов’язково вказати "Обласний центр". При цьому поле "Введіть назву вулиці" стане доступним. Наприклад, за необхідності знайти індекс вулиці Поштової у місті Донецьк, параметри мають бути вказані так

Для усіх інших населених пунктів для пошуку індексів достатньо обрати лише назву населеного пункту.
